# Data Exchange Format Converter

Simple converter converts:</br>

- Gradle & Maven Dependencies</br>
- XML to JSON</br>
- XML to YAML</br>
- JSON to XML</br>
- JSON to YAML</br>
- YAML to XML</br>
- YAML to JSON</br>

Formats an XML file.

# Built With

## Frameworks
